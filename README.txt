ORDY Airdrop Bot Setup Guide

1. Install requirements:
   pip install pyTelegramBotAPI pymysql

2. Create MySQL database and import 'sql_schema.sql'.

3. Edit 'config.py' if needed (token, DB info).

4. Run the bot:
   python3 main.py

Commands:
- /start [ref]
- Use buttons to submit wallet, check status, and claim tokens.
